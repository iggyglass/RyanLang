# RyanLang
A Language to mess with someone of the name Ryan who hates parentheses.

## How to use
Really the language is just a reskin of [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck):

| Token in Brainfuck | Token in RyanLang |
|--------------------|-------------------|
| >                  | (((               |
| <                  | (()               |
| +                  | ()(               |
| -                  | ())               |
| .                  | )((               |
| ,                  | )()               |
| [                  | ))(               |
| ]                  | )))               |
